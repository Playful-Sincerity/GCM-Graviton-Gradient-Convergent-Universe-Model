# GDGM → GCM Rename

Project was renamed on 2026-04-21 from **GDGM — Graviton Density Gradient Mechanics** to **GCM — Graviton Gradient Convergent Universe Model**.

## What Changed

- All in-content references: `GDGM` → `GCM`, `Graviton Density Gradient Mechanics` → `Graviton Gradient Convergent Universe Model`
- Lowercase path segments: `gdgm/` → `gcm/`
- Subfolder rename: `gdgm/` → `gcm/`
- File renames:
  - `gdgm_nbody.py` → `gcm_nbody.py`
  - `gdgm_v2.md` → `gcm_v2.md`
  - `GDGM.code-workspace` → `GCM.code-workspace`

## What Stayed

- The phrase "graviton density gradients" as a description of the underlying mechanism (the physical process of graviton density varying across space), since this refers to the physics, not the project brand.
- Historical references to "GDGM" inside git history and any external citations remain.

## Why

The folder `GCM/` was already the forward-looking label. Internal docs still used the legacy GDGM name, creating mixed branding. Unified under GCM to make the project's identity coherent for papers and outreach.
