## 4. Provenance Trail

Each claim in `theory/`, `STATUS.md`, and `HISTORY.md` should be traceable back through this catalog to either a project-local source or an external source listed above. When in doubt, the chain is:

**README → STATUS / HISTORY → archive-highlights → research/sources/\* → primary source**

If a claim cannot trace back through this chain, it is either editorial consolidation (acknowledge it explicitly) or it is insufficiently grounded (fix it).
