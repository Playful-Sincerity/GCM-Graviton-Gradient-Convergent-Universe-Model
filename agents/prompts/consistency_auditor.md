# Prompt: Consistency Auditor (Wave 1, Living Process)

**Agent**: general-purpose
**Run**: Background
**Date used**: 2026-03-21
**Re-run**: Any time the theory adds new claims or revises existing ones

---

## Purpose
NOT to evaluate whether the theory is correct.
TO ensure whatever the theory claims is internally logically coherent and mathematically consistent.

All postulations may be wrong — that's fine. But they must not contradict each other, and the math must be dimensionally consistent.

## Input
- The four core tenets (axioms, non-negotiable)
- The current set of hypotheses (all revisable)

## Output
Written to: `research/findings/consistency_report.md`

## Audit Checklist (apply to each hypothesis)
1. Dimensional analysis — units consistent on both sides?
2. Circular reasoning — does conclusion assume what it's proving?
3. Contradiction with axioms — does it violate the four core tenets?
4. Contradiction with other hypotheses — does it conflict with another part of the theory?
5. Underspecified mechanism — is this a label masquerading as an explanation?
6. Quantitative match — for any claim to reproduce a known result, does the math actually work?

## Output Sections
1. Critical Issues (must fix)
2. Underspecified Mechanisms (not broken, but not yet a real mechanism)
3. Open Questions (important gaps)
4. What's Internally Sound
5. Recommended Fix Order
